steelhax - A Steeldiver Subwars secondary entrypoint

So yeah this is my first exploit ^^ The installer is going to completely make the exploit ready to use. There's no real menu for the exploit though. However it's nice that the game has SD access. The exploit is going to load the steelhax_payload.bin file from the root of the sd card if you hold b while starting the game. The payload is just an otherapp payload which you can find here. The exploit supports EUR, USA and JPN.

Make sure you have version 1.0 (uninstall updates). Also make sure that you have started the game atleast once and that you choose a mii so that you get into the menu. The game only creates its savefiles at that point. After starting the installer homebrew just confirm everything until you see the firmware selection screen, just enter your firmware and whether you have an o3ds or a new 3ds. If everything successfully installed you just have to start the game and enjoy your homebrew :) If the game says that the savedata is corrupt just retry, I am trying to fix the installer right now. If the game crashes on the other hand it means that the payload file hasn't been found. Try putting the otherapp payload as steelhax_payload.bin on your sd card root and hold b while starting the game.
